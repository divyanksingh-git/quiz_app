# quiz_app

A simple MCQ quiz app with firebase google login.

## Dependencies Used

1. ebase_core: ^2.9.0
2. ebase_auth: ^4.4.0
3. gle_sign_in: ^6.0.2
4. get_storage: ^2.1.1
5. timer_count_down: ^2.2.2
6. percent_indicator: ^4.2.3
